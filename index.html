Q: What is the purpose for Module? The purpose for modules is that you can have animations activate at certain times when the button is pressed.
In this site you will have the option to download the boilerplate code. It will always be updated to the current stable version.
For the time being you can create your own file with Champion Name, their time scale if they have any, and the ability cast.
Every Needs to be filled out for the download button to work;
If you have any issues with the site or something is not working properly please click the issue button; it will direct you to the issue github page.
Q: What type of file will i download? The File type for the boilerplate is C# [C Stash] the input file type is text file.
The github link will direct you to the main project where you can download the software to use modules and animation for LOL & RL.
